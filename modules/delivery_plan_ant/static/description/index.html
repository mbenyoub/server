delivery_plan Module

Add and Edit Details of Delivery

By Elico Corp

This modules adds the 2 new fields to the purchase and sales orders which are: Start Delivery Date and End Delivery Date.
Users will be able to edit delivery details with this module.

Features:

－ Automatically assign Purchase / Sales Orders a Preparation Time Slot and Delivery Time Slot to match Start and End Delivery Date
－ Edit Delivery Time Slot and Assigned Transport and its Drivers for Orders
－ Kanban View of All Deliveries categorized by Delivery Time Slot and Assigned Transport
－ Input reason for returned goods
－ Easier Confirmation for Delivery before Departure by Customer Service
－ Adjust of payments of the received by Customers
－ Input notes for special services for customers during delivery
